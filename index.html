Help

1. Click "Start New Game" button to begin.
2. Swap tiles to complete the Puzzles. Remember that you can only swap a puzzle with its adjacent tile: the nearest left, right, upper or a puzzle below the selected puzzle. To make it easier for you the correct tiles are highlighted so you can easily locate them.
3. Complete the picture of a man in a hat to finish the game. The fewer turns you you will need to complete the task the better. You can put your name down to be shown on Best Results.

The game does not support touch screen functionality, so it is NOT recommended using mobile devices for this game.

Enjoy the game!
